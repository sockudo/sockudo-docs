## WebSocket Endpoint

Sockudo serves WebSocket upgrades on:

```text
/app/{appKey}
```

Pusher-compatible clients usually append:

```text
?protocol=7&client=js&version=<client-version>
```

## Core Events

- `pusher:connection_established`
- `pusher:error`
- `pusher:ping`
- `pusher:pong`
- `pusher_internal:subscription_succeeded`
- `pusher_internal:member_added`
- `pusher_internal:member_removed`

## Sockudo Extension Events

- `pusher:enable_delta_compression`
- `pusher:delta_compression_enabled`
- `pusher:delta`
- `pusher:delta_cache_sync`
- `pusher:delta_sync_error`

## Subscribe Payload Extensions

Sockudo accepts extra subscription fields in `pusher:subscribe` data:

- `filter` / `tags_filter`: filter tree for tag filtering.
- `delta`: per-subscription delta settings.
